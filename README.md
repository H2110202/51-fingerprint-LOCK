# 51-fingerprint-LOCK

![2873526da4a562ac6bf7fc9e3af43156](https://github.com/user-attachments/assets/e592e877-97f2-44ef-9d33-43917092645d)


大一的时候自学51单片机，碰巧作为宿舍长有一个生活中的问题解决：舍友们老是忘带钥匙。所以我想着正好可以用矩形键阵用于输入密码，AS608指纹模块用于识别指纹。
于是，一个全新的宿舍指纹锁就诞生了。
